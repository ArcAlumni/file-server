<!-- GETTING STARTED -->

### Prerequisites

1. Install Java using [this](https://docs.oracle.com/javase/9/install/installation-jdk-and-jre-linux-platforms.htm)

2. Install apache maven using [this](https://maven.apache.org/download.cgi)


### Installation

for linux

    sh make.sh
   
for windows

    make.bat

### Run

for linux

    sh run.sh
   
for windows

    run.bat
